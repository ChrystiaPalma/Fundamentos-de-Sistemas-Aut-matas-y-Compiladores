Fase 1: Análisis (Front-end)

  • 1. Léxico Convierte caracteres en Tokens. Usa Regex/DFA.
  • 2. Sintáctico Valida gramática y crea Árbol AST. Usa GLC y PDA.

Fase 2: Síntesis (Back-end)

  • 4. Código Intermedio Código abstracto agnóstico (ej. Java Bytecode).
  • 5. Optimización Reducción de ciclos, eliminación de código muerto.

Manejo de Errores

Un buen compilador no solo traduce, sino que diagnostica. Los errores pueden ocurrir en cualquier fase:

Léxico
Carácter inválido (ej. @ en C++ fuera de string).
Sintáctico
Falta un punto y coma ; o paréntesis desbalanceado.
Semántico
Tipos incompatibles (ej. int x = "hola").
